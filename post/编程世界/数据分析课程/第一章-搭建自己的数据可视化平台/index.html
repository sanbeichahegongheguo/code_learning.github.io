我已经见了太多分别讲如何数据分析、如何数据可视化以及如何部署数据分析产品的文章了，但是从来没有一个把这些内容融合到一起的，全部都是割裂的。

比如数据分析可能用的是一套工具如python里的pandas、numpy或其他神经网络、机器学习，到了可视化，又有一堆如matlibplot、seaborn、echarts等等，而这之前又都是讲在本地运行，到了如何部署远程又是别的一套工具，让人疲于奔命，每次都觉得自己掌握了一点，但是总是拼不到一块。

前段时间看了一个老师的一个文章

当时还很兴奋，觉得终于有全部融合到一起的工具了吗？

但是我发现这个老师也是个坑，他讲得太简略了。跟着他操作，可能能走一遍流程，但是还有大量的内容他完全不补充说明，当然，像streamlit如何使用，这个是不应该老师讲解，而是自己去看文档。但是其他的设置呢，老师就根本不提了。

所以，我这里，给大家完完整整演示一遍，如果我们想搭建一个数据分析的可视化平台，到底应该怎么做。

首先要知道，我们的根本在于数据分析，然后才是可视化，然后才是建一个放在远程服务器上的平台来让大家访问以满足自己的小小虚荣心。

所以第一个，我们得想想自己要分析啥数据、展示啥数据。

因为老师提供了很好的工具：streamlit，所以我们就基于这个工具来说。

这个工具是用来可视化的，它把 pandas、numpy 经整合很好了，所以我们就基于这个工具来说。

首先是看streamlit的官方说明文档。

第一当然是安装，不过这个很简单：pip install streamlit即可。还会跟着附属安装一堆其他工具，不用管。

然后是工具的使用，我把文档里大概的例子贴出来：

'Starting a long computation...'

# Add a placeholder
latest_iteration = st.empty()
bar = st.progress(0)

for i in range(100):
    # Update the progress bar with each iteration.
    latest_iteration.text(f'Iteration {i+1}')
    bar.progress(i + 1)

'...and now we\'re done!'

st.title('Uber pickups in NYC')

DATE_COLUMN = 'date/time'
DATA_URL = ('https://s3-us-west-2.amazonaws.com/'
            'streamlit-demo-data/uber-raw-data-sep14.csv.gz')


@st.cache
def load_data(nrows):
    data = pd.read_csv(DATA_URL, nrows=nrows)
    lowercase = lambda x: str(x).lower()
    data.rename(lowercase, axis='columns', inplace=True)
    data[DATE_COLUMN] = pd.to_datetime(data[DATE_COLUMN])
    return data

# Create a text element and let the reader know the data is loading.
data_load_state = st.text('Loading data...')
# Load 10,000 rows of data into the dataframe.
data = load_data(100)
# Notify the reader that the data was successfully loaded.
data_load_state.text('Loading data...done!')

st.subheader('Raw data')
st.write('Done! (using st.cache)')

st.subheader('Number of pickups by hour')
hist_values = np.histogram(
    data[DATE_COLUMN].dt.hour, bins=24, range=(0, 24))[0]
st.bar_chart(hist_values)

st.subheader('Map of all pickups')
st.map(data)
hour_to_filter = st.slider('hour', 0, 23, 17)  # min: 0h, max: 23h, default: 17h
filtered_data = data[data[DATE_COLUMN].dt.hour == hour_to_filter]
st.subheader(f'Map of all pickups at {hour_to_filter}:00')
st.map(filtered_data)


if st.checkbox('Show raw data'):
    st.subheader('Raw data > Use a button to toggle data')
    st.write(data)

import streamlit as st
# ①标题
st.title("主标题")
st.subtitle("副标题") # 会比主标题字体小一点

# ②直接显示文字
# 可以把你想显示的文字直接用引号引住写出来即可，如
"显示这段文字"
# 如果是变量，就直接写出变量即可，如
x = '显示这个x包含的字符串'
y = 345
x
y # 文档就会直接显示x和y包含的内容

# ③st.write('这也是显示文字的一种方法')，如官方文档里的例子
x = st.slider('x')
st.write(x, 'squared is', x * x)
